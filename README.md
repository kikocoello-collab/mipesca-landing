# MiPesca — Web de marketing

Landing estática de [MiPesca](https://github.com/kikocoello-collab/APP-PESCA-TOP),
la app de pesca en el mar: condiciones, mareas, solunar, diario de capturas y comunidad.

- `index.html` — página completa (HTML + CSS inline, sin build)
- `img/` — capturas de la app y logo

## Publicar

Es una web 100% estática: sirve cualquier hosting. Con GitHub Pages:
Settings → Pages → Deploy from branch → `main` / raíz.
